# Екзамен: Міні‑проєкт у Jupyter Notebook (60 хв)

Ця тека містить матеріали для проведення іспиту з курсу "Основи генеративного ШІ" (2 курс, Комп’ютерна інженерія).

## Для студентів
- **Основний шаблон для виконання**: `exam_template.ipynb`
  - Заповніть секцію `[0]` (ПІБ, група, варіант, модель, endpoint).
  - Виконайте послідовно секції `[1]` → `[2]` → `[2b]` (чат-режим) → `[3]` → `[4]` (ваш варіант) → `[5]`.
  - У секції `[2b]` чат-режим вимкнено за замовчуванням. Щоб запустити, встановіть `RUN_CHAT = True`. Для безпеки є ліміт `MAX_TURNS`.
  - Ключі — тільки через `.env` (`GITHUB_TOKEN`). Не друкуйте значення ключів у вивід.
- **Іменування файлу перед здачею**: `Прізвище І.Б._varNN.ipynb` (приклад: `Петренко І.О._var07.ipynb`).
- **Завантаження**: на Google‑диск, вказаний викладачем.

## Для викладача
- Опис іспиту: `Іспит_мініпроєкт_GenAI_осінь_2025.md`
- Приклад розв’язку (інструкторський, не роздавати студентам): `exam_variant01_solution_ok.ipynb`
  - Містить реалізацію варіанту 1 (рецепти у строгому JSON), валідацію, авто‑виправлення, параметричні експерименти, а також опційний чат‑режим `[2b]`.

## Зміст теки
- `exam_template.ipynb` — шаблон для студентів (використовує ті ж інструменти/моделі, що в практичних).
- `Іспит_мініпроєкт_GenAI_осінь_2025.md` — офіційне формулювання завдання, структура, критерії.
- `exam_variant01_solution_ok.ipynb` — приклад розв’язку для викладача (інструкторський доступ).

## Вимоги до інструментів
- Endpoint за замовчуванням: `https://models.inference.ai.azure.com`.
- Моделі та SDK — ті, що використані у практичних (наприклад, `azure-ai-inference`, `ChatCompletionsClient`, GitHub Models).
- Використовуйте `.env` для `GITHUB_TOKEN`. Заборонено хардкодити ключі.

## Примітки
- Якщо чат‑режим у Jupyter не приймає введення, переконайтесь, що `RUN_CHAT = True` і середовище підтримує stdin. Альтернатива — використання віджетів (`ipywidgets`).
- За відсутності мережі продемонструйте сформовані промпти, параметри та очікувані формати відповідей.
